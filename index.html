JavaScript Course

3 cores technologies in web development: HTML(content), CSS(presentation) and JavaScript(dynamic effects/programming)

React and Angular are 100% based on JavaScript

HTML(content): NOUNS <p></p> means "paragraph"
CSS(presentation): ADJECTIVES p {color: red;} means "the paragraph text is red"
JavaScript(dynamic effects/programming): VERBS p.hide(); means "hide the paragraph"

Data types: number (float), string, boolean, undefined (does not have a value yet) and null

Dynamic typing (use just var)
